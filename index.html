字蛛：让网页自由引入中文字体成为可能

@font-face {
  font-family: 'xiaofeng';
  src: url('css/xiaofeng.woff') 
  format('woff');
}
.f-fmy {
  font-family: 'xiaofeng';
}
html {
  font-size: 10px;
}
.f-size {
  font-size: 20px;
}
.f-size0 {
  font-size: 120%;
}
.f-size1 {
  font-size: 1.2em;
}
.f-size2 {
  font-size: 2.4rem;
}
.f-size3 {
  font-size: smaller;
}
.f-size4 {
  font-size: larger;
}
